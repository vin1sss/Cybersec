# The Digital Forensics Codex

Este repositório compila o **The Digital Forensics Codex** completo, reunindo tanto os capítulos fundamentais quanto os complementos inspirados em estudos orientados e pesquisas independentes.  O objetivo é oferecer um guia **didático, abrangente e atualizado** sobre forense digital — desde os conceitos básicos até tópicos avançados de análise de sistemas de arquivos e artefatos específicos em diferentes plataformas (Windows, macOS, Linux e redes).

## Estrutura de capítulos

Cada capítulo está organizado em um arquivo Markdown independente.  Use o índice abaixo para navegar e estudar os temas conforme sua necessidade.

| **15 – Tópicos avançados e estudos de caso** → [15_additional_topics.md](15_additional_topics.md) | Abrange temas diversos: comparação entre partições MBR e GPT, análise dos sistemas de arquivos Ext3/Ext4 (extents, timestamps em nanosegundos, journaling), metodologia de análise de aplicações (como Microsoft Teams), técnicas de **live forensics** em Linux e orientações para estudos de caso e investigações sem supervisão. |

Todos os capítulos se complementam para formar um compêndio autossuficiente e robusto, com foco em **didática e profundidade**.  As referências bibliográficas utilizadas são indicadas dentro de cada capítulo através de marcadores de citação (por exemplo, `【123456†L1-L3】`) que apontam para as fontes consultadas.  Esses marcadores devem permanecer no texto para garantir a rastreabilidade das informações.
